TripApp

Back-end project w/ PostgreSQL and Google Maps API

This is a full-stack group project created for my bootcamp. As the team lead on this project, I was responsible for managing the workload for my team members while assisting where needed.

This project integrates with the Google Maps API to pull places of interest based on specific geographical areas. Users then create accounts, and then log into their accounts to save places of interest to a travel list.

User login credentials and travel lists are stored via a PostgreSQL database.
